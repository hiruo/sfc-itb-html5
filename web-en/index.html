About This Web Site

This web site is the lecture materials for the web design in the IT Literacy Course of SFC.
Please refer to this site for personal reference.

This lecture would proceed assuming that all students are able to use a text editor, and are able to touch-type. Students who are unconfident about these skills are recommended to practice beforehand.
