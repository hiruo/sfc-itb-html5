HTML Entities (Special Characters and Escapes)

When coding web pages with HTML, we have learned that each tag begins with a "<" symbol, and ends with a ">" symbol. However, what should do when you want to use such symbols in your page as a text? In this section, we would discuss about such issues.

Special Characters in HTML

In HTML, the following characters are recognized as special characters. The W3C HTML Validator would return errors, if these characters were to appear in the HTML file as regular text unrelated to tags.

  • < : A symbol indicating the beginning of a tag
  • > : A symbol indicating the end of a tag
  • & : A symbol indicating escapes

The following sample shows an example of special characters used in a wrong way. 

  <p>
    HTML<HyperText Markup Language> 
    is a language for creating web pages.
  </p>

In this sample, the symbols, "<" and ">" are used as characters for a text string. Us humans would be able to recognize these as "brackets". However, web browsers would see them as indicators of HTML tags. When browsers detect the first "<" symbol, it would then look for the next ">" symbol, and recognizes the characters in between as HTML tags. In this particular case, browsers would recognize the characters, "HyperText Markup Language", as a HTML tag. Since this sample code does not have an end tag marked "</Hypertext Markup Language>", the validator would return errors.

So, what should we do if we want to use special characters as regular text in HTML??

The Escape Character "&"

The solution to the question above are "escapes". Escapes enables us to express certain characters with alternative codes. This feature is not only supported by HTML, but also other programming languages as well, though rules and formats may not be the same.

In HTML, the ampersand symbol (&) is used for indicating escapes. For example, the "<" symbol, the ">" symbol, and the "&" may be expressed in the following way.

  • < : &lt;
  • > : &gt;
  • & : &amp;

The following code shows an example of the correct way to use special characters within regular text contents. 

  <p>
    HTML &lt;HyperText Markup Language&gt;
    is a language for creating web pages. Language
  </p>
